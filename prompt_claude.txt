## A

### `Avatar:addActiveTitle(title_id)`
> Adds an active title to the avatar.

```lua
player:addActiveTitle(5)  -- Add title ID 5 as active
```

Avatar:addBankBalance(balance)

Adds balance to the avatar's bank account.

```lua
player:addBankBalance(100)  -- Add 100 World Locks to bank
```

Avatar:addBlessing(blessingID)

Adds a blessing to the avatar.

```lua
player:addBlessing(1)  -- Add blessing ID 1
```

Avatar:addCoins(coins)

Adds coins to the avatar.

```lua
player:addCoins(1000)  -- Add 1000 coins
```

Avatar:addFriend(second_player)

Adds a friend to the avatar's friend list (bidirectional).

```lua
local target = getPlayer(123)
player:addFriend(target)  -- Add target as friend
```

Avatar:addGems(gems, instantSet, progressQuests)

Adds gems to the avatar and updates the display.

```lua
player:addGems(500, 1, 1)  -- Add 500 gems, instant update, progress quests
```

Avatar:addGrowpassPoints(points)

Adds Growpass points to the avatar.

```lua
player:addGrowpassPoints(100)  -- Add 100 Growpass points
```

Avatar:addLevel(level)

Adds levels to the avatar.

```lua
player:addLevel(5)  -- Add 5 levels
```

Avatar:addMail()

Adds mail to the avatar's mailbox.

```lua
player:addMail({
    sender = "System",
    subject = "Welcome!",
    content = "Hello and welcome to the server!"
})
```

Avatar:addMod(modID, value)

Adds a mod to the avatar.

```lua
player:addMod(-77, 60)  -- Add Vanish mod for 60 seconds
```

Avatar:addNote(note)

Adds a staff note to the avatar's account.

```lua
player:addNote("Suspicious activity detected")
```

Avatar:addPlaytime(time)

Adds playtime to the avatar.

```lua
player:addPlaytime(3600)  -- Add 1 hour of playtime
```

Avatar:addSubscription(Type, exp)

Adds a subscription to the avatar.

```lua
player:addSubscription(1, os.time() + 86400)  -- Add Super Supporter for 1 day
```

Avatar:addTitle(title_id)

Adds a title to the avatar's collection.

```lua
player:addTitle(10)  -- Add title ID 10
```

Avatar:adjustBlockHitCount(adjust)

Adjusts the block hit count for the avatar (-10 to +10).

```lua
player:adjustBlockHitCount(-10)  -- All blocks break in 1 hit
```
